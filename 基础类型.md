# 基础类型

## undefined 和 null

都是定义了之后，只能定义为自身

这两个类型是其他类型的子类型

## never 类型

表示永远不存在的类型

永远也没有办法完全执行完

```js
const errorFun = (message: string): never => {
  throw new Error(message);
};
```

## 静态类型

2. 编辑器可以方便的提示类型的一些方法

## 函数

```js
const fn = (str: string): string => {

```

## 数组和元组 tuple

```js
// 数组
const arr: (number | string)[] = [1, "2", 3];

// 元组 长度是已知，固定的
const arr1: [number, string, number] = [1, "2", 1];
```
